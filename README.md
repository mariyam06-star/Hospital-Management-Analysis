# Hospital-Management-Analysis
Description  The Hospital Management Dashboard is an interactive Power BI project designed to analyze hospital operations efficiently. It provides insights into patient records, doctor workload, diagnosis trends, billing analysis, insurance coverage, and bed occupancy.  
The report helps hospital administrators and healthcare teams monitor key performance indicators (KPIs) and make data-driven decisions.

Bed Occupancy Analysis

Patient-Level Filtering

Interactive Visualizations

Dataset Information
The dataset contains hospital-related information such as:
| Column Name             | Description               |
| ----------------------- | ------------------------- |
| Patient_ID              | Unique patient identifier |
| Admit_Date              | Patient admission date    |
| Discharge_Date          | Patient discharge date    |
| Followup_Date           | Follow-up schedule        |
| Doctor_Name             | Assigned doctor           |
| Diagnosis               | Patient diagnosis         |
| Billing_Amount          | Total hospital billing    |
| Health_Insurance_Amount | Insurance covered amount  |
| Bed_Occupancy           | Bed category used         |
| Department              | Hospital department       |
